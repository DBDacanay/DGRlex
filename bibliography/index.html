Sources used

The Tłı̨chǫ Online Dictionary draws entries from many different sources. The source for each entry is written below the headword.

Most of the entries in the online dictionary are taken from existing written sources, which are listed by shorthand (see list below). However, a large number of entries have also been provided directly for the dictionary by fluent speakers. In these cases, the source is listed as the name of the fluent speaker, followed by the year (in brackets) when they provided the word (e.g. Rosa Mantla (2024)).

The major written sources used in the dictionary are listed below, in rough order of the number of entries provided by each. The speakers who contributed to each source are listed below each reference:
